Matter — You Are Not Alone
Matter is a mental health support application for teenagers and young adults experiencing bullying, loneliness, and emotional difficulties.

Problem
Many young people struggle with:
bullying
loneliness
lack of emotional support
In most cases, there is no safe space to talk about their problems.

Solution
Matter provides a safe digital environment where users can:
talk with an AI for emotional support
listen to calming music
use mini-games for distraction
complete emotional self-assessment surveys
connect with people in similar situations
Key Features
user authentication
AI support chat
psychological surveys
user matching system
word mini-game
background music player
user profile system
Why It Matters

Matter helps users:
reduce stress and anxiety
get emotional support
feel less alone
Architecture
User → Frontend (React) → REST API → Backend → Supabase
Tech Stack
Frontend: React
Backend: REST API
Database: Supabase
Platforms: Android / Web
